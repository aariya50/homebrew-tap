# homebrew-tap

Homebrew formulae for [OpenClaw Toggle](https://github.com/aariya50/openclaw-toggle) and related tools.

## Installation

```bash
brew tap aariya50/tap
brew install openclaw-toggle
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `openclaw-toggle` | macOS menu bar app to monitor and control OpenClaw node and SSH tunnel services |
